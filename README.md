# Inventory Management System

## Overview

This project is an **Inventory Management System** built using **ASP.NET Core MVC**.
It is designed to manage inventory records through basic **CRUD operations** such as adding, viewing, updating, and deleting items.

The project follows the **Model-View-Controller (MVC)** architecture, which keeps the application organized and easy to understand.

---

## Features

* Add new inventory items
* View all inventory records
* Update existing item details
* Delete inventory records
* Organized MVC project structure

---

## Technologies Used

* **C#**
* **ASP.NET Core MVC**
* **.NET**
* **Razor Views (CSHTML)**
* **Visual Studio**

---

## Project Structure

* **Models** - Contains the data models used in the application
* **Views** - Contains Razor pages for the user interface
* **Controllers** - Handles requests and application logic

---

## How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/inventory-management-system-mvc.git
   ```

2. Open the project in **Visual Studio**

3. Restore the required dependencies

4. Run the application by pressing `F5` or clicking **Start**

# Inventory Management System

An ASP.NET MVC web application for managing inventory, products, stock, and related business operations.

## Features

- Product management
- Category management
- Stock tracking
- Add, edit, delete records
- User-friendly dashboard
- Database integration
- Inventory monitoring and updates

## Technologies Used

- ASP.NET MVC
- C#
- Entity Framework
- SQL Server
- HTML
- CSS
- JavaScript
- Bootstrap

## Project Structure

- `Controllers/` - application controllers
- `Models/` - data models
- `Views/` - UI pages
- `Content/` - CSS and static files
- `Scripts/` - JavaScript files
- `Migrations/` - database migration files

## How to Run

1. Open the project in Visual Studio.
2. Restore NuGet packages.
3. Update the database connection string in `Web.config`.
4. Run the database migrations if needed.
5. Press `F5` or click **Start** to run the project.

## Requirements

- Visual Studio
- SQL Server
- .NET Framework
- NuGet Package Manager

## Purpose

This project is developed for learning and managing inventory operations digitally in a simple and efficient way.
